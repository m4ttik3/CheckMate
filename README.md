# 📱 CheckMate — Приложение для официанта

> Мобильный помощник для фиксации заказов гостей с интеграцией базы данных.  
> Проект выполнен в рамках производственной практики. Замена бумажных блокнотов и минимизация ошибок при передаче заказов на кухню.

## 🎯 Цель проекта

Разработать клиент-серверное приложение, которое позволяет официанту:
- Вести учёт столов в зале
- Принимать заказы c учётом комментариев
- Отправлять заказы в базу данных и на кухню

## 🧩 Функциональные требования (MVP)

### 👤 Роли
| Роль | Возможности |
|------|--------------|
| **Официант** | Логин, просмотр столов, создание/редактирование заказов, отправка на кухню |
| **Администратор** | Управление столом, редактирование меню, просмотр выручки |

### 🪑 Работа со столами
- Выбор номера стола
- Статусы столов:
  - 🟢 Свободен
  - 🟡 Обслуживается
  - 🔴 Ждёт оплаты

### 📝 Создание заказа
- Выбор блюд из БД (название, цена)
- Учёт количества (`+` / `-`)
- Комментарии к позиции (*"без лука"*, *"горячий чай"*)

## 🗄️ Структура базы данных (SQL)
### Описание таблиц

#### `employees`
| Колонка | Тип | Описание |
|---------|------|-------------|
| id | INT | ID сотрудника |
| full_name | NVARCHAR(MAX) | Имя официанта |
| role | INT | `Оффициант` / `Администратор` |
| login | NVARCHAR(50) | Логин |
| password | NVARCHAR(50) | Пароль |
#### `tables`
| Колонка | Тип | Описание |
|---------|------|-------------|
| id | INT | Номер стола |
| status | NVARCHAR(50) | `Свободен` / `Обслуживается` / `Ждет оплату` |

#### `menu`
| Колонка | Тип | Описание |
|---------|------|-------------|
| id | INT | ID блюда |
| category_id | INT | Категория |
| name | NVARCHAR(MAX)| Название |
| price | MONEY | Цена в рублях |
| is_available | NVARCHAR(50) | В наличии? |

#### `orders`
| Колонка | Тип | Описание |
|---------|------|-------------|
| id | INT | Номер заказа |
| table_id | INT (FK→tables) | Стол |
| menu_id | INT | Блюдо |
| quantity | INT | Количество |
| comment | NVARCHAR(MAX) | Комментарий к заказу |
| total | MONEY | Сумма к оплате |

#### `category`
| Колонка | Тип | Описание |
|---------|------|-------------|
| id | INT | ID Категории |
| category_name | NVARCHAR(MAX) | Категория |

#### `role`
| Колонка | Тип | Описание |
|---------|------|-------------|
| id | INT | ID Роли |
| role | INT | Роль |
